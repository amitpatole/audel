# Audel — Ears for AI Agents

A machine-graded **audio** feedback loop that coding agents consume to self-correct before
claiming *done*: **play → hear → report → fix.**

This Space is a static landing page. Use Audel from Python or the CLI:

```bash
pip install audel
audel demo          # FAIL → PASS, no API key required
```

- **Docs:** https://amitpatole.github.io/audel/
- **Source:** https://github.com/amitpatole/audel
- **PyPI:** https://pypi.org/project/audel/

Part of the eyes/ears/brain trio with [AgentVision](https://huggingface.co/spaces/amitpatole/AgentVision)
and [Verel](https://huggingface.co/spaces/amitpatole/verel), sharing one contract,
[agentsensory](https://pypi.org/project/agentsensory/).
